Testing gulp build tool

To build the project:

1. install all npm packages by running npm install, if not installed;
2. modify the connection parameter in app.js to point to the correct web api server
3. run gulp clean;
4. run gulp build;
5. run gulp minifyJs;
6. change the main.js in index.html file to main.min.js (make suer main.min.js has been successfully generated in the folder of js);
7. test.
